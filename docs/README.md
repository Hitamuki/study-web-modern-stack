# Docs

このディレクトリにはシステム全体の設計やドメインモデルに関するドキュメントを格納します。

## 収録内容

- `context-map.md` - バウンデッドコンテキストと各システム/サービスの連携を示すコンテキストマップ
- `domain-model.md` - ドメインエンティティ・値オブジェクト・集約境界などを示すドメインモデル図
- `er-diagram.md` - データベースの主要テーブルとリレーションを示すER図
