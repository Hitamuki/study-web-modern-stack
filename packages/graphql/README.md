# @repo/graphql

## 概要
モノレポ内の各フロントエンド（Web, Mobile, Desktop）で共通して使用する GraphQL クライアント、型定義、および Codegen 設定を管理するパッケージです。

## 主な役割
- **一括型生成**: 各アプリ（`apps/*`）のソースコードをスキャンし、一箇所で全ての GraphQL 型定義を生成します。
- **共通 ApolloProvider**: Apollo Client の初期化ロジックを共通化し、各アプリで一貫した接続設定を提供します。
- **共通 `gql` 関数**: `graphql-codegen` の Client Preset を使用した、型安全な `gql` タグを提供します。

## 主要コマンド

### 型定義の更新 (Codegen)
Hasura のスキーマ変更や、各アプリでのクエリ変更を反映させるために実行します。
```bash
# プロジェクトルートから実行
pnpm --filter @repo/graphql codegen
```

## 運用上の重要事項

### 型更新後の各アプリへの影響
このパッケージはモノレポ内のすべてのフロントエンドで共有されています。そのため、API の変更（Hasura スキーマの更新など）に伴って `codegen` を実行すると、**すべてのアプリに対して同時に型変更が適用**されます。

型更新を行った後は、必ず以下の各アプリでビルドエラーや型の不整合が発生していないかを確認し、必要に応じてコードを修正してください。

- **`apps/web`**
- **`apps/mobile`**
- **`apps/desktop`** (追加時)

このように一箇所で管理することで、API 変更に対するフロントエンド全体の修正漏れを防ぎ、常に最新かつ正確な型定義で開発を進めることができます。
